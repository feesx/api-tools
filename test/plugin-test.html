JSON Formatter 插件功能测试

测试说明：此页面包含各种JSON格式的测试数据，用于验证Chrome插件的功能是否正常工作。

测试步骤：

  • 确保Chrome浏览器已加载插件
  • 使用浏览器右键菜单或插件图标测试格式化功能
  • 验证插件是否能正确格式化不同类型的JSON数据
  • 测试插件在无效JSON情况下的错误处理

测试类型 1 - 简单JSON对象

{ "name": "张三", "age": 30, "email": "zhangsan@example.com", "isStudent": false, "score": 85.5 }

测试类型 2 - JSON数组

[ { "id": 1, "name": "张三", "age": 30, "email": "zhangsan@example.com" }, { "id": 2, "name": "李四", "age": 25, "email": "lisi@example.com" }, { "id": 3, "name": "王五", "age": 35, "email": "wangwu@example.com" } ]

测试类型 3 - 嵌套JSON结构

{ "company": "Tech Corp", "address": { "street": "123 Main St", "city": "Beijing", "country": "China" }, "employees": [ { "name": "张三", "role": "Developer", "skills": ["JavaScript", "Python", "React"] }, { "name": "李四", "role": "Designer", "skills": ["Photoshop", "Sketch", "Illustrator"] } ], "projects": [ { "name": "Project X", "status": "active", "budget": 100000 }, { "name": "Project Y", "status": "completed", "budget": 50000 } ] }

测试类型 4 - 特殊字符

{ "message": "测试包含\n换行符\n的字符串", "url": "https://example.com/path/to/resource?param1=value1¶m2=value2", "html": "
\n

标题

\n

内容

\n
" }

测试类型 5 - 无效JSON

{ "name": "张三", "age": 30, "email": "zhangsan@example.com" "isStudent": false // 这里缺少逗号 }
