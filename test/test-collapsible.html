JSON Formatter 折叠功能测试

测试步骤：

  1. 点击浏览器工具栏中的 JSON Formatter 图标
  2. 复制下面的复杂嵌套JSON示例
  3. 粘贴到左侧输入框
  4. 点击"格式化"按钮
  5. 测试折叠功能：
    • 点击对象或数组前的 ▶ 图标进行折叠/展开
    • 点击"展开全部"按钮展开所有节点
    • 点击"折叠全部"按钮折叠所有节点
