## 📁 Project Overview

API Tools is a lightweight Chrome browser extension that provides:
- **API Testing** (similar to Postman)
- **JSON Formatting** with syntax highlighting and validation
- **XML Formatting** with syntax highlighting and validation

## 🚀 Key Features

- **API Testing**: Support for GET, POST, PUT, DELETE, PATCH requests with real-time response statistics
- **JSON Formatting**: Auto-formatting, syntax highlighting, error validation, collapsible view
- **XML Formatting**: Auto-formatting, syntax highlighting, error validation
- **Language Support**: Chinese and English interfaces
- **Request History**: Save and manage previous API requests

## 📦 Installation

1. Open Chrome browser and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked"
4. Select the project's `src/` folder
5. The extension is installed successfully

## 🎯 Quick Start

1. Click the API Tools icon in the Chrome toolbar
2. Select "Open Formatter" to access all features
3. Choose the desired functionality tab (Postman, JSON, or XML)
